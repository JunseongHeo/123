# 서블릿과 JSP의 기초 개념

서블릿 servlet은 server + applet의 합성어로 자바를 이용하여 웹에서 실행되는 프로그램을 작성하는 기술을 말한다.

서블릿은 자바 클래스 형태의 웹 애플리케이션을 말하는데, 
브라우저를 통해 자바 클래스가 실행되도록하는 HttpServlet 클래스를 상속받아 구현한다.
